RoadRunner项目和场景系统

在 RoadRunner 中，一个项目包含由多个 RoadRunner 场景共享的资产。您可以在同一个项目中创建多个场景，场景可以在项目中共享资源。
项目资产包括从 3D 模型、纹理贴图和矢量图形等文件创建的各种组件。它们还包括特定于 RoadRunner 的各种文件，例如材料和道路标记样式。

项目

当您运行RoadRunner时，您必须选择或创建一个项目。当前项目显示在标题栏上。当前项目始终是 RoadRunner 中的活动项目。

创建新项目

  1. 打开RoadRunner应用程序，然后在开始页面的Scene选项卡上，单击New Scene。
  2. 在“选择项目”窗口中，单击“新建项目”，然后浏览到要在其中创建项目的文件夹。要在文件夹中创建项目，该文件夹必须为空。笔记：如果您在网络驱动器上创建项目，对资产文件所做的更改可能不会自动反映在库浏览器中。此外，性能可能会变慢。为了提高性能和完整的Library Browser功能，请在本地磁盘上创建一个项目。
  3. 选择要与项目一起安装的资产。默认情况下， RoadRunner项目包括少量材料、模型和其他资产。您还可以单独购买RoadRunner Asset Library Add-On，其中包含大量通用和特定国家/地区的资产。
  • 如果您拥有RoadRunner Asset Library许可证，请单击“是”Assets以使用库资产填充文件夹。
  • 如果您没有许可证，则此选项将被禁用。相反，单击否以使用RoadRunnerAssets中包含的一组默认资产填充该 文件夹。
RoadRunner将打开一个在新项目中创建的新场景。可以从创建项目的文件夹访问项目内容。

笔记

建议您将整个项目文件夹置于版本控制之下。有关详细信息，请参阅项目和场景版本控制 部分。
项目文件夹内容
项目文件夹包含以下子文件夹：
  • Project— 此文件夹包含一个名为 的文件 Project.rrproj，它为项目定义了唯一的引用 ID。不要修改或移动此文件。
  • Scenes– 此文件夹存储使用此项目的各个场景。首次创建项目时，此文件夹包含多个示例场景。当您为当前项目保存新场景时，RoadRunner默认到此文件夹。
  • Exports– 此文件夹是从RoadRunner写出导出数据的默认位置。当您第一次创建项目时，此文件夹是空的。您不需要将此文件夹用于导出的数据。有关导出数据的更多信息，请参阅导出场景。
  • Assets– 此文件夹存储可在场景中使用的所有资产文件。该文件夹及其子文件夹出现在库浏览器中。对于树中的每个资产Assets，RoadRunner 会自动创建一个具有 .rrmeta 文件扩展名的关联图元文件。图元文件包含与资产相关的附加数据，其详细信息因不同的资产类型而异。图元文件还包含一个唯一 ID，您可以使用它来识别特定资产。即使您重命名资产或将其移动到 Assets 文件夹树中的不同文件夹中，此 ID 也不会改变。始终将此图元文件保存在与资产本身相同的位置。使用库浏览器执行的资产操作会自动更新相应的图元文件。有关使用资产的更多详细信息，请参阅创建、导入和修改资产。

更改当前项目

如果您已经在 RoadRunner 中打开了一个项目，要切换到一个新项目，请从菜单栏中选择文件 ，然后选择更改项目。然后，在 Select a Project 窗口中，创建一个新项目，浏览不同项目的根文件夹，或从Recent Projects列表中选择一个最近打开的项目。

保存项目

要保存项目，请从菜单栏中选择文件，然后选择保存项目。此外，保存场景还会保存您对当前项目所做的任何更改。在保存当前项目之前，不会保存修改的资产。

场景

场景文件包含一个区域，其中包含道路、表面、道具和其他场景方面等对象。它是 RoadRunner 中编辑的主要文件类型。场景可以表示从小区域（例如单个十字路口）到大区域（例如城市的一部分）的任何事物。一个场景可以包含多条道路、交叉路口、道路标记、道具、地形部分等。
单个场景保存为 .rrscene 文件，通常保存在 Scenes 项目的文件夹中。您可以在同一个项目中创建多个场景，场景可以在项目中共享资源。
RoadRunner 在任何给定时间只有一个场景处于活动状态。当前场景的名称显示在标题栏中。如果场景尚未保存，标题栏将场景名称显示为 New Scene。
如果当前场景中有未保存的更改，RoadRunner 会提示您在开始新场景、加载现有场景或退出程序之前保存当前场景。

创建新场景

要在现有场景中工作时创建新场景，请在菜单栏上选择 文件，然后选择新建场景。或者，按Ctrl+N。
要从起始页创建新场景，请执行以下步骤：
  1. 在开始页面的 Scene选项卡上，单击New Scene。
  2. 选择以前的项目或单击“新建项目”以在新项目中创建场景。

打开现有场景

要打开最近打开的现有场景，请直接从 File下的Recent Scenes菜单 打开它。或者，选择 文件，然后选择打开，或按Ctrl+O，然后浏览您要打开的场景。
如果所选场景文件位于Scenes项目文件夹的子文件夹中，则会自动加载该项目。如果您的场景保存在其他地方，场景将存储项目目录的相对路径。
如果RoadRunner无法找到或加载项目，软件会提供选择以前项目或浏览其他项目的选项。

保存场景

要保存场景，请从菜单栏中选择文件，然后选择保存场景。保存场景还会保存您对当前项目所做的任何更改。在保存当前项目之前，不会保存修改的资产。

项目和场景版本控制

当多人对项目资产或场景进行更改时，使用 Git™ 等版本控制系统来控制文件版本。使用修订控制系统，请遵循以下策略：
  • 使用版本控制系统管理项目树中的文件，包括图元文件 (.rrmeta)。
  • 不要合并任何 RoadRunner 内部文件，因为它们是二进制文件。
  • 确保任何时候只有一个用户对任何特定文件进行更改。
