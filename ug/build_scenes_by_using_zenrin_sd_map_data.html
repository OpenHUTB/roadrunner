通过选择链接和节点浏览导入的数据。您可以在 上查看其属性 属性窗格。在 SD 地图中选择的道路元素类型 场景编辑画布确定可用属性。
简单链接
  • ID — 所选链路的唯一标识号。
  • 在构建期间跳过 — 指定是添加还是跳过此链接 在构建过程中。如果选择此属性，SD Map 将此链接表示为虚线，并且在构建过程中将忽略该链接。自 在构建过程中包括链接，将其显示为实线，清除此链接 属性。SD 地图查看器工具导入禁用了“构建期间跳过”属性的实际链接，并将其显示为实体 线。SD 地图数据可能包含同一区域不支持的链接。默认情况下，SD 地图查看器工具会在启用“构建期间跳过”属性的情况下导入这些不受支持的链接，并在 SD Map 中以虚线形式显示它们。您可以看到检测到的列表 “输出”窗口中不支持的链接 ID。选择这些 ID 以 导航到并查看 SD 地图中的链接。
注意
您可以单击并拖动以选择矩形区域内的多个链接 利息。您也可以按住 Shift 并单击其他链接以添加 他们到选择。对于“属性”窗格中的所有选定链接，可以集中控制“生成期间跳过”属性。
  • 道路宽度（以米为单位）— 最小和最大道路宽度。
  • 车道数 — 前进和后向车道数。前进和后退属性显示最小值和最大值（如果地图数据指定） 它。

每个链接都有多个控制点，每个控制点都包含指定其的位置属性 （X，Y，Z）位置。

  • 连接链接 — 显示连接到 选定的节点。每个连接的链接都标有其关联的 ID 和 取向。

修路

您可以使用其中一个或多个过程为导入的数据构建道路。
  • 所有数据 — 构建所有导入的数据。
  • 选择链接 — 单击并拖动以选择矩形区域内的链接 利息。
您还可以删除选定的链接以避免构建它们。
对于此示例，请勿选择任何链接。
然后，单击左侧工具栏上的“建设道路”按钮 场景编辑画布。
在简单地图构建器 对话框中，您可以查看和修改以下选项：
  • 保留高度 - 默认情况下，SD 地图查看器 工具可保留导入道路的高度。要删除高度信息， 清除此选项。
  • 清除现有数据的场景 — 默认情况下，SD 地图 查看器工具在用于 构建场景。要保留场景中的现有道路，请清除此选项。
  • 驾驶侧 — 默认情况下，SD 地图查看器工具将道路左侧视为前进的行驶方向。考虑右侧 的道路作为前进的行驶方向，从下拉列表中选择。Right
  • 启用重叠组 - 默认情况下，SD 地图查看器 工具不会在道路重叠处创建自动交汇点。要创建交汇点， 该工具使用导入的 SD 地图数据中指定的显式交汇点信息。自 在几何重叠处创建自动交汇点，清除此选项。欲了解更多信息 有关重叠组的信息，请参见防止在道路之间创建自动交汇点。选择“启用重叠组”属性时，该工具 默认情况下，将“重叠组名”属性设置为 。您可以使用重叠组 名称属性，用于控制在以下情况下自动创建交汇点的行为 在现有场景上构建 SD 地图数据。例如，如果道路在现有的 场景的“重叠组”属性值为 ，并且您不希望创建自动交汇点 在它们与 SD 地图数据指定的道路之间存在几何重叠时，必须将重叠组名称属性设置为 。否则，该工具将在以下位置创建自动交汇点 现有场景的道路与指定的道路之间的几何重叠 导入的 SD 地图数据。SceneBuildTransferImportTransferImport
  • 自动检测桥梁 — 默认情况下，SD 地图查看器 当道路具有不同时，工具会在道路交叉口创建桥梁 海拔。该工具将桥梁两侧延伸 20 米 路口。您可以通过更改网桥来更改扩展量 跨度通货膨胀值。要防止该工具创建桥梁，请清除此 选择。有关详细信息，请参阅道路建设 工具。

通过单击全部构建在整个场景中构建道路。如果需要帮助， 要仅构建场景中的道路子集，请选择要包含在场景中的链接，然后 单击“生成选定项”。
修路后，可以在路跑者中修改场景。您还可以将场景导出到 ASAM OpenDRIVE 文件。
如果RoadRunner在修建道路时检测到车道标记重叠，则可能会显示此标记 SD 地图构建器结果对话框中的消息：

要解决此问题，请打开道路规划工具，单击并导航到重叠处 位置和调整道路中心。

排查导入和构建问题

问题：建成的道路在交叉路口包含道路之间的间隙。
方法：打开自定义交汇点工具，导航到受影响的 交汇点，然后在属性窗格中增加射线距离。
问题：已建成的道路在道路网格中包含陡峭的上升或下降 结。
方法：打开拐角工具，导航到受影响的交汇点， ，然后减小属性窗格中的拐角半径。
问题：已建成场景包含地形下的道路。
方法：打开表面工具，导航到受影响的位置， 并手动调整地形。
