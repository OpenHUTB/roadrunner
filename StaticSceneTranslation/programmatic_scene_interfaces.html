编程场景接口

RoadRunner 提供用于执行常见工作流任务的编程接口 例如打开、关闭和保存场景和项目以及导入和导出 场景。
  • 如果您使用的是 MATLAB，则可以使用 MATLAB 函数来执行这些操作。这些功能 需要自动驾驶工具箱™许可证。
  • 如果您使用的是外部编程语言，则可以 使用 RoadRunner gRPC API 执行这些操作。您可以将 API 编译为支持的任何语言 通过 gRPC，例如 C++ 或 Python。

类别

  • 用于场景和高清地图的 MATLAB 函数使用 MATLAB 导入和导出场景（需要自动驾驶工具箱）
  • 适用于场景和高清地图的 gRPC API使用 C++、Python 和 gRPC 框架支持的其他编程语言导入和导出场景
