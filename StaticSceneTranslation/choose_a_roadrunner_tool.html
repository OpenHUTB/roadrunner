选择RoadRunner工具

RoadRunner 工具栏包含用于编辑场景的各种工具。您的工具选择确定可以在场景编辑画布中选择和编辑的对象。在 此外，某些操作会自动更改活动工具。例如，拖动 某些类型的资源从库浏览器添加到场景中 该资产到场景中，并自动将RoadRunner切换到该资产的适当工具。

道路工具

道路规划工具
创建和铺设道路。
公路环线工具
建造闭环路，例如用于创建环形交叉路口。
截面工具
操纵道路横截面处的倾斜、加冠和路缘形状。
道路高度工具
操纵道路的垂直剖面图。
道路超高工具
调整超高（坡度和倾斜角度）以获得全宽路。
路斩工具
将单条道路切成两条相连的道路。
路施工工具
指定路段的实际构造。
道路限速工具
沿路段设置限速。
支路 工具
创建入口匝道、出口匝道和道路分割。
道路偏移 工具
调整两条端到端道路之间的连接。
道路锚工具
定义道路锚点以在场景中放置方案（需要 RoadRunner 方案)

交汇点工具

习惯 交汇点工具
覆盖RoadRunner自动路口功能以实现高级 例。
角落 工具
调整连接角的形状和材料。
结 曲面工具
控制道路标高和横截面如何影响 交叉 口。
机动 工具
操纵交汇点内的单个机动道路（路径）。
信号 工具
配置交汇点信号化和信号流量相位。

车道工具

车道 工具
删除车道并更新车道类型和行驶方向。
车道宽度 工具
调整沿道路的车道宽度。
车道偏移 工具
调整道路中心车道的位置。
人行道高度工具
修改人行道和路缘高度。
车道添加 工具
沿道路添加完全成型的车道。
车道形式 工具
沿道路添加成型车道或终点车道。
巷雕 工具
在车道中创建锥形切口，例如用于在中位数创建专用转弯车道。
车道斩 工具
将单车道分成两条车道。
车道分割 工具
纵向将车道拆分为两条车道。

打标工具

车道标记 工具
在路面上放置点标记（模具），例如箭头和文字。
标记点 工具
在路面上放置点标记（模具），例如箭头和文字。
标记曲线 工具
在任意位置放置直线或弯曲标记。
标记 多边形工具
定义沥青补丁区域或在道路和地形上重复标记条纹 表面。
交通孤岛工具
创建自由形式的流量孤岛
停车 工具
定义停车位和其他与停车相关的标记。
人行横道和停车线工具
在十字路口的拐角之间添加人行横道和停车线

道具工具

道具点 工具
在场景中放置单个道具并将它们连接到其他道具。
道具曲线 工具
沿自由曲面曲线放置道具和延伸。
道具多边形 工具
将道具放置在任意形状的区域内。
道具跨度 工具
沿路放置道具和挤压件。
标志 工具
修改自定义标志，例如街道名称标志和高速公路广告牌。

地形工具

表面 工具
对道路周围的表面进行建模，例如人行道、车道、停车场和自然曲面 地形。

地理信息系统工具

空中 影像工具
管理航空影像文件的导入和配置。
高程图 工具
管理数字高程模型 （DEM） 文件的导入和配置。
点云 工具
管理激光雷达点云文件的导入和配置。
矢量数据 工具
管理矢量数据文件的导入和配置，并浏览形状属性。
OpenDRIVE 查看器工具
可视化ASAM OpenDRIVE数据以进行导入。
道路 CRG 工具
管理ASAM OpenCRG文件的导入和配置。
场景生成器 工具
从高清地图数据生成 3D 道路模型。
世界 设置工具
配置环境模型的地理位置和大小以进行数据导入和 出口。
SD 地图查看器 工具
从 Zenrin Japan Map API 3.3（Itsumo NAVI API 0.3）数据生成 0D 道路模型。

实用工具

测量 工具
测量场景中的位置、距离和角度。
截图 工具
生成并保存当前相机视图的图像。
开放驱动器导出预览工具
可视化和验证场景的 ASAM OpenDRIVE 导出并加载外部 ASAM OpenDRIVE 文件。
