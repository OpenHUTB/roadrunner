Vector Data Tool

描述

矢量数据工具管理矢量的导入和配置 数据文件，并启用对形状属性的浏览。RoadRunner可以加载各种地理配准矢量地图文件，例如形状文件 （）、OpenStreetMap 文件 （）、GeoJSON 文件 （、） 和 GPS Exchange 文件 （）。矢量数据 例如点、线和多边形可以从这些文件及其关联的文件中加载 属性。.shp®.osm.json.geojson.gpx
有关支持的格式列表，请参阅矢量数据资产页面。
注意
矢量数据工具用于可视化和探索矢量文件类型 用作视觉参考。不支持将矢量地图数据自动转换为 RoadRunner 内部格式。

打开矢量数据工具

在 RoadRunner 工具栏上，单击矢量数据工具按钮：

例子

导入地理配准矢量地图
  1. 单击矢量数据工具按钮。
  2. 在库浏览器中，导航到包含以下内容的目录 要导入的矢量数据资产。有关更多详细信息，请参阅矢量数据 资产。
  3. 单击资源并将其从库浏览器拖动到 3D 中 现场。
注意
如果尚未为此场景设置地理位置，则场景中心设置为 影像的纬度和纵向中心。您可以使用以下方法更改场景中心 世界设置工具。
如果已设置地理位置，但导入的图像在 场景的最大范围，将出现一个错误对话框并取消导入。
移除矢量地图
  1. 单击矢量数据工具按钮。
  2. 在要删除的矢量地图的边界框中单击。不要单击 矢量元素本身。而是单击边界框中的空白区域。当 单击地图，边界框变为红色。
  3. 按 Delete 键或从菜单栏中选择编辑>删除。
调整矢量贴图的属性
  1. 单击矢量数据工具按钮。
  2. 在要调整的矢量地图的边界框内单击。不要单击 矢量元素本身。而是单击边界框中的空白区域。当 单击地图，边界框变为红色。
  3. 根据需要在属性窗格上调整任何属性。
切换矢量数据的显示
选择菜单栏上的“查看 矢量数据>或按 F7 键。
在矢量地图中查看要素的属性
  1. 单击矢量数据工具按钮。
  2. 单击要检查的矢量要素。要素的属性为 显示在属性窗格中。
  • 您可以选择多个要素来查看 所有选定的要素。
  • 如果单击属性窗格中的属性字段名称或值，主场景将突出显示对象 该值。
基于属性的颜色特征
您可以根据类型或其他属性值将自定义颜色应用于矢量要素 如下：
  1. 单击矢量数据工具按钮。
  2. 选择要重新着色的要素（或按 Ctrl+A 键 全选）。
  3. 在属性窗格中，找到您 想要用于重新着色。如果单击属性字段名称或值，则主要 场景突出显示具有该值的对象。
  4. 右键单击属性字段名称或值，然后选择颜色 与。
  5. 在颜色选取器中选择一种颜色。

清晰的特征颜色
如果为矢量要素分配了自定义颜色，则可以将自定义颜色清除为 遵循：
  1. 单击矢量数据工具按钮。
  2. 选择要重置其颜色的特征（或按 Ctrl+A 全选）。
  3. 在属性窗格中，找到其 要重置的颜色。
  4. 右键单击属性字段名称或值，然后选择清除 颜色。
