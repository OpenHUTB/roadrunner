RoadRunner 的键盘快捷键和鼠标操作

编辑

这些操作适用于编辑窗口，在某些情况下适用于窗格，例如 2D 编辑器、库浏览器和属性窗格。
任务 行动
撤消或重做。 Ctrl+Z 或 Ctrl+Y
全选。 Ctrl+A

对象选择和操作

任务 行动
选择一个对象。 左键单击
将对象添加到选区 按住 Shift，然后点按对象。
从选区中删除对象。 按住 Ctrl+Shift，然后单击对象。
选择多个对象。 按住 Shift，然后左键单击并拖动以绘制 框，
翻译对象。 选择一个或多个对象，然后按 T 键 显示翻译工具。
旋转对象。 选择一个或多个对象，然后按 R 键 显示旋转工具。

相机控制（编辑画布）

任务 行动
围绕兴趣点旋转相机。 按住 Alt，然后单击并拖动。
放大或缩小。 按住 Alt，向上移动鼠标滚轮或 下。
沿 x方向或 y方向平移整个场景 按住 Alt，左键单击，然后右键单击， 然后拖动
在 z 方向上升高和降低摄像机。按住 Alt+Shift，左键单击和右键单击， 然后拖动
围绕选定对象构图相机。 选择一个对象并按下 F
切换到透视图，变小。 按 P。
切换到正交视图 按 O。

更新 Linux Ubuntu 键映射

在 Linux 中，Ubuntu 16.04，按 Alt 键移动当前窗口，并且 按 Windows 键会显示某些帮助叠加层。推荐 您更新 Ubuntu 以使用 Windows 键（而不是 Alt 键）来移动窗口。若要进行此更新，请按照以下操作 步骤：
  1. 如果尚未安装，请安装。在 命令行中，输入以下代码：dconf-editorsudo apt-get install dconf-editor
  2. 打开。dconf-editor
  3. 导航到 gnome > gnome > wm > 首选项>组织。
  4. 将 更改为 。mouse-button-modifier<Super>
注意
将有效密钥分配给 非常重要。 将该选项留空可防止鼠标与任何鼠标交互 窗户。mouse-button-modifier
