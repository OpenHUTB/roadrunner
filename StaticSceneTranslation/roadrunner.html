roadrunner

描述

对象使您能够执行常见的工作流任务 在 RoadRunner 应用程序中，例如打开、关闭和保存场景和项目，从 MATLAB 命令行。您还可以使用对象函数从文件导入数据，以及 将场景从RoadRunner导出为其他格式。roadrunner
首次创建对象之前，必须 安装RoadRunner并以交互方式激活您的RoadRunner许可证。roadrunner
该对象需要自动驾驶工具箱™的许可证。roadrunner

创造

语法
rrApp = roadrunner(projectFolder)
rrApp = roadrunner(projectFolder,Name=Value)
描述
rrApp = roadrunner（projectFolder） 从默认安装文件夹位置启动 RoadRunner，并在 位于 指定位置的现有项目。projectFolder
rrApp = roadrunner（projectFolder，Name=Value） 也使用 name-value 参数来设置对象的属性。roadrunner

例子

从默认安装文件夹启动RoadRunner应用程序
使用该变量指定现有项目的路径。例如，此代码显示 Windows® 计算机上位于 的项目路径。projectFolderC:\RR\MyProject
projectFolder = "C:\RR\MyProject";
创建一个对象，并通过将项目指定为要创建场景的位置来打开 RoadRunner。此示例假定 RoadRunner 安装在其默认位置。roadrunner
rrApp = roadrunner(projectFolder);
更改默认 RoadRunner 安装文件夹设置
使用 MATLAB 函数修改默认的 RoadRunner 安装文件夹设置。settings
您可以使用该函数为跨 MATLAB 会话保留的对象的属性指定一个值，也可以为给定的 MATLAB 会话指定一个值。InstallationFolderroadrunnersettings
要为 设置永久默认值，请编辑 的值。指定的值在单个用户的多个 MATLAB 会话中保持不变。InstallationFolder property PersonalValue
s = settings;
s.roadrunner.application.InstallationFolder.PersonalValue = "C:\Program Files\RoadRunner R2022a\bin\win64";
还可以为属性设置临时默认值。指定的值仅保留当前 MATLAB 会话，并在会话结束时清除。InstallationFolder
s = settings;
s.roadrunner.application.InstallationFolder.TemporaryValue = "C:\MyRoadRunner Install\bin\win64";
同时启动多个 RoadRunner 应用程序实例
通过创建对象的多个实例来打开多个 RoadRunner 应用程序。您可以使用每个实例以编程方式与它打开的 RoadRunner 应用程序进行交互。默认情况下，此代码中使用的场景包含在 RoadRunner 项目中。roadrunner
通过指定项目打开 RoadRunner 的第一个实例。在此示例中，项目位于路径 上。Lane KeepC:\RR\Lane Keep
rrApp1 = roadrunner("C:\RR\Lane Keep");
在第一个项目中打开场景。FourWaySignal
openScene(rrApp1,"FourWaySignal.rrscene");
通过指定项目打开 RoadRunner 的第二个实例。在此示例中，项目位于路径 上。USCityC:\RR\USCity
rrApp2 = roadrunner("C:\RR\USCity");
在第二个项目中打开场景SanAntonio
openScene(rrApp2,"SanAntonio.rrscene");
在控制台模式下启动 RoadRunner 应用程序
使用该变量指定现有项目的路径。例如，此代码显示 Windows® 计算机上位于 的项目路径。projectFolderC:\RR\MyProject
projectFolder = "C:\RR\MyProject";
创建一个对象，并通过将项目指定为要创建场景的位置来打开 RoadRunner。此示例假定 RoadRunner 安装在其默认位置。指定属性以使用非图形终端在控制台模式下启动应用程序。roadrunnerNoDisplay
rrApp = roadrunner(projectFolder,InstallationFolder="C:\Program Files\RoadRunner R2022b\bin\win64",NoDisplay=true);
