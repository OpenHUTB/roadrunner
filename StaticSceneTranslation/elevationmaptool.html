Elevation Map Tool

描述

高程地图工具用于管理 数字高程模型 （DEM） 文件。RoadRunner可以从各种文件格式导入高程图数据，例如DEM，IMG， JPEG 2000 和 TIFF。其中一些格式支持地理配准，并且可以自动 相应地定位。
有关支持的文件类型列表，请参阅高程地图资产页面。
可以为一个区域导入多个高程图以提供全面覆盖。这可以 导致某些导入的地图在某些区域中重叠。您可以调整优先级 每个地图，以确定哪一个在重叠区域中优先。

打开高程地图工具

在 RoadRunner 工具栏上，单击高程地图工具按钮：

例子

导入地理配准高程地图
  1. 单击高程地图工具按钮。
  2. 在库浏览器中，导航到包含以下内容的目录 高程地图 要导入的资产。
  3. 右键单击资产并确保默认类型为 设置为 。Elevation Map
  4. 单击资源并将其从库浏览器拖动到 场景编辑画布。
注意
如果尚未为此场景设置地理位置，则场景中心设置为 影像的纬度和纵向中心。您可以使用以下方法更改场景中心 世界设置工具。如果已设置地理位置，但导入的图像在 场景的最大范围，将出现一个错误对话框并取消导入。
导入非地理配准高程地图
要在RoadRunner中正确显示高程图，程序必须知道如何在地球上定位地图。为了最好的 精度，获取包含地理位置信息的高程数据（通过使用格式 例如 GeoTIFF 或 JPEG 2000）强烈建议使用。有关以下方面的链接和示例 获取与RoadRunner兼容的GIS数据，请参见下载GIS数据以在RoadRunner中使用。
如果您的高程图没有地理位置信息，则可以手动 使用以下步骤设置地理位置信息。
如果您已经知道投影
如果您已经知道要使用的特定投影（即，您有一个“proj”或 “wkt”投影字符串），您可以在文件上进行设置，如下所示：
  1. 单击高程地图工具按钮。
  2. 在库浏览器中，导航到包含以下内容的目录 要导入的高程地图文件。
  3. 右键单击文件资产，并确保 默认 类型设置为 。Elevation Map
  4. 单击属性窗格中的设置自定义投影按钮。
  5. 将 或 字符串粘贴到 文本字段。'proj''wkt'
  6. 单击“确定”。
  7. 通过调整分辨率以匹配 高程地图的米/像素。
8.单击并拖动库中的高程地图资源 浏览器进入 3D 场景。
如果您不知道投影
以下步骤使您能够使用任意灰度图像进行高程，例如 作为单独应用程序的屏幕截图。但是，结果不会很高 准确。
如果你不知道投影，你可以尝试不同的投影 文件上的值。这些说明将横轴墨卡托投影应用于文件。
  1. 单击高程地图工具按钮。
  2. 在库浏览器中，导航到包含以下内容的目录 要导入的图像文件。
  3. 右键单击文件资产，并确保 默认 类型设置为 。Elevation Map
  4. 单击属性窗格中的设置自定义投影按钮。
  5. 确定高程数据中心点的纬度和经度。 然后，将自定义投影窗口中的纬度和经度值调整为 火柴。
6.单击在以下位置使用横轴墨卡托。然后，单击“确定”。
7.通过调整分辨率以匹配 米/图像的像素。
8.单击并拖动库中的高程地图资源 浏览器进入 3D 场景。
移除高程地图
  1. 单击高程地图工具按钮。
  2. 单击要删除的高程地图。
  3. 按 Delete 键或从菜单栏中选择编辑>删除。
调整高程图的属性
  1. 单击高程地图工具按钮。
  2. 单击要编辑的高程地图。
  3. 通过属性窗格根据需要调整高程地图属性。
注意
当多个高程地图在一个位置重叠时，系统需要 决定使用哪一个。选择高程图并单击推送到 属性窗格中的底部或移至顶部按钮可调整地图的优先级以解析 重叠。
切换高程图的显示
从菜单栏中选择“查看>高程”，或按 F5。
将道路和其他对象投影到高程图
许多 RoadRunner 对象可以投影到高程地图表面。具体步骤是 记录在相应的工具中。例如，将道路投影到 高程图可在此处找到：项目道路 高程图。
在大多数情况下，步骤是相同的：
  1. 选择适当的工具。
  2. 在场景编辑画布中再选择一个对象。
  3. 单击左侧工具栏中的相应投影按钮。
在表面三角测量中包括高程图
默认情况下，高程图显示为白色地形表面。这 表面不包括在导出的场景中。
要在场景中包括高程地图的视觉影响，必须创建 覆盖该区域的地形表面，然后启用全局高程。更多详情 在地形曲面上，请参阅路面在 RoadRunner 中的工作原理。
有关详细信息，请参阅控制是否 表面使用高程样本。
