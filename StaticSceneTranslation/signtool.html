信号工具

描述

使用Sign Tool 修改自定义标志，如街道名称标志和高速公路广告牌。您可以将标志划分为矩形区域，然后在标志中放置文本、图形和彩色矩形。

打开签名工具

在 RoadRunner 工具栏上，单击“Sign Tool ”按钮：

例子

在场景中放置标志实例

要在场景中放置一个标志，请单击并将标志资源从库浏览器拖到场景中。或者，遵循以下步骤:
  1. 单击Sign Tool按钮。
  2. 单击所需的标志资产。
  3. 在场景中单击鼠标右键以放置标志。

修改符号区域

您可以在符号中创建单独的区域并单独修改每个区域。

将一个区域拆分为两个区域

  1. 单击签名工具按钮。
  2. 单击要编辑的标志：资产浏览器中的标志资产或 3D 场景中的标志实例。
  3. 单击要拆分的区域。
  4. 单击添加水平拆分或添加垂直拆分按钮。
单击该按钮时，区域将在中间分割并绘制分隔线。若要调整分隔符的位置，可以单击并拖动分隔符本身到适当的位置。

删除区域

  1. 单击签名工具按钮。
  2. 单击要编辑的标志。
  3. 单击要删除的区域。
  4. 按删除以删除该区域。

编辑区域的颜色和边界属性

  1. 单击签名工具按钮。
  2. 单击要编辑的标志。
  3. 单击要编辑的区域。
  4. 在属性窗格中设置颜色和边界属性。

修改标志文本

您可以向标志添加文本框并编辑其中的文本。
您还可以修改文本的可视属性并修改字体。RoadRunner使用通过操作系统安装的标准系统字体。

添加新文本框

  1. 单击签名工具按钮。
  2. 单击要编辑的标志：资产浏览器中的标志资产或 3D 场景中的标志实例。
  3. 单击“ Add Text”按钮，在标志中创建一个新的可编辑文本框。
文本框继承上次选取的文本框的属性并显示 居中于最后一个选取的区域。

删除文本框

  1. 单击签名工具按钮。
  2. 单击要编辑的标志：资产浏览器中的标志资产或 3D 场景中的标志实例。
  3. 单击要删除的文本框。
  4. 按 Delete 键，或从菜单栏中选择编辑>删除。

编辑文本框中的文本

  1. 单击签名工具按钮。
  2. 单击要编辑的标志：资产浏览器中的标志资产或 3D 场景中的标志实例。
  3. 单击要编辑的文本框。
  4. 在“属性”窗格中选择“文本”属性，然后键入所需的字符串。

更改文本框的字体或其他可视属性

  1. 单击签名工具按钮。
  2. 单击要编辑的标志：资产浏览器中的标志资产或 3D 场景中的标志实例。
  3. 单击要编辑的文本框。文本框的属性显示在 属性窗格。
  4. 在属性窗格中编辑所需属性。“小型大写字母”选项可用于将小写字母的格式设置为 较小的大写字母

修改标志图形

您可以向标志添加位图或矢量图形，并修改这些图形的形状、颜色或视觉属性。

添加新的标志图形

  1. 单击签名工具按钮。
  2. 点击你想要编辑的标志:在资产浏览器中的一个标志资源或在3D场景中的一个标志实例。
  3. 单击图形并将其拖动到标志上。

删除标志图形

  1. 单击签名工具按钮。
  2. 单击要编辑的标志：资产浏览器中的标志资产或 3D 场景中的标志实例。
  3. 单击要删除的图形。
  4. 按 Delete 键或从菜单栏中选择编辑>删除。

编辑图形形状

  1. 单击签名工具按钮。
  2. 单击要编辑的标志：资产浏览器中的标志资产或 3D 场景中的标志实例。
  3. 单击要编辑的图形。
  4. 单击并拖动图形的边缘。指针变为调整大小符号。

更改图形的颜色或其他可视属性

  1. 单击签名工具按钮。
  2. 单击要编辑的标志：资产浏览器中的标志资产或 3D 场景中的标志实例。
  3. 单击要编辑的图形。图形的属性将显示在属性窗格中。
  4. 在属性窗格中编辑所需的属性。

修改符号矩形

您可以在符号内添加矩形，并修改矩形的形状、颜色和其他视觉属性。

添加新的标志矩形

  1. 单击签名工具按钮。
  2. 单击要编辑的标志：资产浏览器中的标志资产或 3D 场景中的标志实例。
  3. 单击左侧工具栏上的“Add Rectangle”按钮。
新矩形继承上一个选定矩形的属性，如果没有选定矩形，则具有默认值。

删除标志矩形

  1. 单击签名工具按钮。
  2. 单击要编辑的标志：资产浏览器中的标志资产或 3D 场景中的标志实例。
  3. 单击要删除的矩形。
  4. 按 Delete 键或从菜单栏中选择编辑>删除。

编辑矩形的形状

  1. 单击签名工具按钮。
  2. 单击要编辑的标志：资产浏览器中的标志资产或 3D 场景中的标志实例。
  3. 单击要编辑的矩形。
  4. 单击并拖动图形的边缘。指针变为调整大小符号。

更改矩形的颜色或其他可视属性

  1. 单击签名工具按钮。
  2. 单击要编辑的标志：资产浏览器中的标志资产或 3D 场景中的标志实例。
  3. 单击要编辑的矩形。矩形的属性显示在属性窗格。
  4. 在属性窗格中编辑所需的属性。
