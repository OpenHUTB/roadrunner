MATLAB实验室场景和高清地图功能

RoadRunner 提供 MATLAB 函数，用于执行常见的工作流任务，例如打开、 关闭、保存场景和项目以及导入和导出场景。 这些函数通过 MATLAB 对象进行操作，该对象打开并设置 RoadRunner 主用户界面，以便您可以通过编程方式控制它。
RoadRunner 还提供了 MATLAB 函数，可将自定义数据转换为 RoadRunner HD Map 数据模型，并将您的数据导入 RoadRunner。用 这些功能，您可以创建新地图，编辑现有地图，并读取和 从文件写入。您还可以表示车道、车道边界、车道标记、 高清地图中的交汇点、障碍和标志。
这些功能需要自动驾驶工具箱许可证。

对象

roadrunner 使用 MATLAB 启动 RoadRunner 应用程序
roadrunnerHDMap 使用 MATLAB 创建 RoadRunner 高清地图

功能

连接、关闭和状态
close 使用 MATLAB 关闭 RoadRunner
roadrunner.connect 使用 MATLAB 连接到 RoadRunner 的开放实例
status 使用 MATLAB 获取 RoadRunner 的当前状态
项目
newProject 使用 MATLAB 创建新的 RoadRunner 项目
openProject 使用 MATLAB 的 Open RoadRunner 项目
saveProject 使用 MATLAB 保存 RoadRunner 项目
现场
newScene使用 MATLAB 创建新的 RoadRunner 场景
openScene使用 MATLAB 打开 RoadRunner 场景
saveScene使用 MATLAB 保存 RoadRunner 场景
导出和导入
exportScene使用 MATLAB 导出 RoadRunner 场景
exportCustomFormat使用 MATLAB 将 RoadRunner 场景导出为自定义格式
importScene使用 MATLAB 将场景导入 RoadRunner
高清地图
write使用 MATLAB 将 HD 地图写入二进制文件
read使用 MATLAB 从二进制文件中读取高清地图
plot使用 MATLAB 绘制RoadRunner高清地图
readCRS使用 MATLAB 从 RoadRunner 高清地图读取坐标参考系统 （CRS） 数据

主题

在控制台模式下以编程方式控制 RoadRunner
在非图形环境中使用终端启动RoadRunner。
使用 MATLAB 导出多个场景
使用 MATLAB 函数将项目中的所有 RoadRunner 场景导出为支持的格式。
使用 MATLAB 函数在格式之间转换场景
导入RoadRunner项目中的多个场景，并使用MATLAB函数将它们导出为支持的格式。
使用 RoadRunner 高清地图以编程方式构建简单道路
使用RoadRunner HD Map MATLAB函数构建简单的道路。
使用 RoadRunner 高清地图构建派克峰RoadRunner 3D 场景（自动驾驶工具箱）
使用 MATLAB 函数为科罗拉多州派克峰构建 RoadRunner 3D 场景。
