导入场景数据

使用导入的 GIS 数据、ASAM OpenDRIVE 道路网络或高清地图数据构建场景®
您可以使用导入到 RoadRunner 中的数据来构建对真实位置进行建模的场景。例如，RoadRunner使您能够导入地理信息系统（GIS）数据，例如 作为高程图或航空影像。添加时使用此数据作为视觉参考 通往场景的道路。要将 GIS 数据或其他资产导入 RoadRunner，请在库浏览器中拖动文件。
您还可以导入ASAM OpenDRIVE文件，在RoadRunner中修改场景，并将场景导出回ASAM OpenDRIVE。
使用 SD 地图查看器工具，您可以通过导入数据来构建场景 来自标准清晰度 （SD） 地图服务，例如 Zenrin Japan Map API 3.0（Itsumo NAVI API 3.0）。
此外，使用场景生成器工具，您可以通过以下方式构建场景： 从高清 （HD） 地图服务（如 HERE HD 实时地图）导入数据，以及 汤姆汤姆高清地图。使用此工具需要RoadRunner Scene Builder附加组件。
要开始使用导入的 GIS 数据，请使用世界设置工具 设置场景的地理限制。

工具

World Settings Tool 配置环境模型的地理位置和大小以进行数据导入和出口
Aerial Imagery Tool 管理航空影像文件的导入和配置
OpenDRIVE Viewer Tool 可视化 ASAM OpenDRIVE 数据以进行导入
Elevation Map Tool 管理数字高程模型 （DEM） 文件的导入和配置
Point Cloud Tool 管理激光雷达点云文件的导入和配置
Vector Data Tool 管理矢量数据文件的导入和配置并探索形状属性
Road CRG Tool 应用和可视化路面数据
Scene Builder Tool 从高清地图数据生成 3D 场景
SD Map Viewer Tool 使用 SD 地图服务生成 3D 场景

资产
